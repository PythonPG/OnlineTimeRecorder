# テーブル設計

**社員テーブル**

| 列英語名 | 列名   | 説明       |
| :----:   | :----: |:---:       |
| phone | 電話番号 | 基本的に携帯番号 |
| password | パスワード　| 管理者も見れます。
| name     | 氏名   | 社員の氏名 |
| corporation_name | 会社名 | 社員が所属する会社の名前 |
| date | 日付　| 入力する日付
| expire_date | 日付 | 今日の日付|
| job_start | 出社時刻 | 今日の出社時刻 | 
| job_end | 退社時刻 | 今日の退社時刻 |
| status | 状態 | 出社、休暇、退社など |
| min | 規定最小 | 140など |
| max | 規定最大 | 180など |
| overtime | 残業時間 | 残業時間 |
